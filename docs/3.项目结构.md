## 2.项目结构

一个项目，在最外层他们应该是一样的，至少需要有:

|       文件夹        |     说明     |
| :--------------: | :--------: |
|       docs       |   项目文档说明   |
|     pro_name     |    项目名称    |
|      tests       |    测试用例    |
|    README.md     |    项目介绍    |
| requirements.txt | 该项目依赖的第三方库 |

那接下来需要讨论的，就是 `pro_name` 的内部结构该是什么样的呢
